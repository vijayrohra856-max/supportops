# SupportOps – Interactive Product Experience Platform

A modern SaaS-inspired collaboration dashboard built with **React** and **Vite**. This project focuses on creating a highly interactive, state-driven user experience with advanced UI patterns, smooth animations, and reusable components.

---

## Overview

SupportOps simulates a modern collaborative workspace where users can manage tasks, organize work using a Kanban board, monitor team activity, and interact with a responsive, feature-rich interface. The project emphasizes frontend architecture, state management, animations, and user experience rather than backend functionality.

---

## Features

- Multi-panel dashboard layout
- Responsive sidebar navigation
- Interactive Kanban board
- Drag and Drop functionality
- Global state management
- Dark / Light theme with persistence
- Command Palette (Ctrl + K)
- Activity panel
- Team management interface
- Presence indicators
- Optimistic UI updates
- Undo / Redo functionality
- Skeleton loading states
- Smooth page transitions
- Micro-interactions and animations
- Responsive design
- Local Storage persistence

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/SupportOps.git
```

### Navigate to the Project Folder

```bash
cd SupportOps
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will run locally at:

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
